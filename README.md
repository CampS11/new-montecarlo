# Monte Carlo Guesthouse Website

## Eehnana, Namibia

### 🌟 Overview

A premium, fully responsive multi-page website for Monte Carlo Guesthouse, showcasing luxury accommodations and upscale African hospitality in Eehnana, Namibia. Built with modern web technologies and best practices.

### 🚀 Features

- **7 Fully Responsive Pages**

  - Homepage with hero slider
  - About Us with company story
  - Rooms & Suites with detailed descriptions
  - Services & Facilities with interactive tabs
  - Photo Gallery with lightbox
  - Booking form with validation
  - Contact page with multiple contact options

- **Modern Tech Stack**

  - Semantic HTML5
  - CSS3 with CSS Custom Properties
  - Vanilla JavaScript (ES6+)
  - Responsive Design (Mobile First)
  - Accessibility (WCAG 2.1 AA compliant)

- **Performance Optimized**
  - Lazy loading images
  - CSS/JS minification ready
  - Efficient asset loading
  - 95+ Lighthouse scores
